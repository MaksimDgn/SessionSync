How to use
Press CTRL + SHIFT + S to open the panel or use the widget provided to open the panel
You can change the default hotkeys in add-on settings
Tooltips
Context menu
Press left click to access aditional context options
Bookmark menu
Open, Copy URL, Edit or Remove bookmarks from Sessions
URL preview
Move your mouse above a bookmark to view the URL.
Click on the URL bar to copy the link
Edit panel
Edit session or bookmark info. Press Enter to save or Esc to cancel
Create empty session
Start from scratch? No problem... just create an empty session and add bookmarks later
